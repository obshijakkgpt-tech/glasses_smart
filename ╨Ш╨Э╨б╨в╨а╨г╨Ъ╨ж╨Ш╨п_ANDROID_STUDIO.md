# Как собрать APK в Android Studio (по шагам)

У тебя уже установлена Android Studio. Этот путь — локальный (на твоём ПК).
Если кажется сложным — используй GitHub (см. ИНСТРУКЦИЯ_GITHUB.md), там проще.

---

## Шаг 1. Распакуй проект

Распакуй `CyanBridge-redesign.zip` (правый клик → «Извлечь всё»).
Запомни, куда — например `C:\Users\ТЫ\Desktop\cyanbridge-redesign`.

---

## Шаг 2. Открой проект

1. Запусти Android Studio.
2. На стартовом экране нажми **Open** (или File → Open).
3. Выбери распакованную папку **целиком** (ту, внутри которой лежат `app`,
   `gradle`, `build.gradle.kts`) и нажми **OK**.
4. Если спросит про доверие проекту — **Trust Project**.

---

## Шаг 3. Подожди синхронизацию (Gradle Sync)

1. Внизу появится полоса прогресса «Gradle Sync» / «Downloading…».
   Android Studio сам скачает Gradle, Android SDK и зависимости.
2. **Первый раз это долго** — 5–15 минут, зависит от интернета. Это нормально.
3. Если вверху появится жёлтая плашка «Install missing SDK / Accept licenses» —
   нажми на ссылку и согласись (Accept). Studio доставит недостающее.
4. Дождись надписи «Gradle sync finished» (внизу).

---

## Шаг 4. Собери APK

1. Верхнее меню: **Build** → **Build App Bundle(s) / APK(s)** → **Build APK(s)**.
2. Подожди сборку (внизу прогресс). В конце справа внизу всплывёт окошко
   «APK(s) generated successfully».
3. Нажми в этом окошке ссылку **locate** — откроется папка с файлом
   `app-debug.apk`.
   (Путь обычно: `app\build\outputs\apk\debug\app-debug.apk`.)

---

## Шаг 5. Установи на телефон

**Проще всего:** перекинь `app-debug.apk` на телефон (кабель / Telegram /
облако), открой его на телефоне и установи. Разреши «установку из неизвестных
источников», если попросит.

**Или через USB:** подключи телефон кабелем, включи на нём «Отладку по USB»
(Настройки → Для разработчиков; если раздела нет — 7 раз тапни «Номер сборки»
в разделе «О телефоне»). В Android Studio вверху выбери телефон и нажми
зелёную кнопку ▶ Run — приложение само установится и запустится.

---

## Если что-то пошло не так

- **Sync долго «висит»** — это часто просто медленная закачка SDK, подожди.
- **Красная ошибка про SDK/licenses** — обычно решается нажатием на ссылку в
  жёлтой/красной плашке вверху и согласием (Accept).
- **Не помогло** — скопируй текст ошибки (нижняя панель «Build») и пришли мне
  в чат, разберёмся.
